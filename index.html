presentacion bootstrap

Estamos aprendiendo bootstrap, una manera de maquetar utilizando 12 columnas a lo largo de toda la pantalla, no importa el tamaño que tenga. se puede hacer totalmente responsive mediante esta forma de trabajo.

paso 1

En la pagina de bootstrap, fuimos a la version 4.6 y copiamos el link de css en el head, sin crear por el momento un archivo css en nuestro repositorio

paso 2

luego copiamos los links "script" que estan en la seccion separate en la pagina y los pegamos justo antes que termine el body. todo esto que copiamos aca es el CDN

paso 3

en el html creamos un div class="container-fluid" abuelo para que ocupe el 100% de la pantalla, un div class row padre para que se muestre en una fila y dentro de ese mismo otros cuatro divs hijos con la class col-sm-3 para que cada uno de esos divs ocupen 3 columnas cada uno en el tamaño sm

paso 4

ahora hay que hacerlo responsive: a cada div agregarle la clase col-"tamaño"-"cantidad" que define la cantidad de columnas segun el tamaño- col-sm-6 quiere decir que cuando esta en tamaño sm, cada div ocupe 6columnas, o sea el 50%... col-md-3 = en tamaño medium c/div ocupa 3col (25%). tamaño xs toma cada div como 12col (100%) por defecto.
